About Jitter

Jitter is a simple stop motion animation tool. You can save your animations as animated GIFs, PNGs containing all the frames, or a zipped folder of PNGs.

Keyboard Shortcuts

The following keyboard shortcuts are available.

  • tab Toggle UI
  • d Toggle Storyboard view (in progress)
  • ⌃+z/⌘+z Undo (per-frame)
  • ⌃+y/⇧+⌘+z Redo (per-frame)
  • n New File
  • ⌃+s/⌘+s Save File (as SVG)
  • ⌃+o/⌘+o Load File (from SVG)
  • g Save as GIF
  • p Save as PNG
  • ⇧+1 Select Pen tool
  • ⇧+2 Select Rotate tool
  • ⇧+3 Select Move tool
  • ⇧+4 Select Zoom In tool
  • ⇧+5 Select Zoom Out tool
  • ⇧+6 Select Eraser tool
  • + Increase pen or eraser size
  • - Decrease pen or eraser size
  • 1-8 Change color to match palette #
  • ⇧+n Add new frame
  • ⇧+⌦/⇧+⌫ Delete frame
  • ⇧+c Copy Frame
  • ⇧+x Clear Frame
  • ⇧+← First Frame
  • ← Prev Frame
  • → Next Frame
  • ⇧+→ Last Frame
  • k Toggle Onionskin
  • r Run animation
